### Install latest python
```shell script
$ brew install python
```

### .zshrc or .bash_profile for OS X 10.12++
```shell script
export PATH="/usr/local/opt/python/libexec/bin:$PATH"
```

### Open Pyton3 Console
```shell script
$ phyton3 
# Use exit() or Ctrl-D to exit
```
